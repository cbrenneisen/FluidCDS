## Setup Locally

Make sure Node.js & npm are properly installed.

Installing all dependencies (npm modules, React, React-Router, RxJS, etc):
```
npm install
```

Converting all scss to css and launching the app:
```
npm start
```

App runs on localhost:3000 by default
